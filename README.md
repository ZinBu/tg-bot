# tg-bot
Микро-бот для Telegram, который рассылает сообщения во все чаты, в которых находится.
Реализован в виде микросервера, который можно задеплоить на Heroku и через него общаться API телеграм.

Пример URL:
```
POST https://domen/send
```
Параметры:
```
{
  "text": "Сообщение",
  "token": "токен от TG API, указанный в настройках",
  "chat": "Необязательный парамерт с указанием конктретного чата"
}
```
